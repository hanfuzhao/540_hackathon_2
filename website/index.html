The Problem

Abuse that's
invisible to classifiers

Standard sentiment analysis says "that sounds positive." But victims of gaslighting know the words can twist reality while appearing perfectly polite. Sarcasm, passive aggression, and subtle manipulation fly under the radar — leaving people doubting themselves.

Gaslight Guard blends zero-shot NLP with a curated pattern library to surface what conventional models miss — and arms users with real, immediate support.

Detection Labels

Four patterns,
one classifier

Each message is scored across all four categories simultaneously — catching complex, overlapping patterns that single-label classifiers miss.

✅
Sincere

Genuine, healthy communication. Direct language with no hidden negative intent.

"I appreciate you sharing that. It sounds really difficult."
😏
Sarcastic

Positive wording with hidden negative intent. The surface meaning is the opposite of what's communicated.

"Oh great, another meeting that could've been an email."
😤
Passive-Aggressive

Indirect hostility masked as politeness. Compliance on the surface, resistance underneath.

"Fine. Do whatever you want. I'm sure you know best."
🔴
Gaslighting

Language designed to distort the victim's reality — denying events, questioning memory, dismissing feelings.

"You're imagining things. No one else thinks there's a problem here."

How it works

NLP pipeline,
explained

01
Input text

Paste a single message or a full conversation thread. Up to 5 messages can be analyzed together for context.

02
Zero-shot scoring

facebook/bart-large-mnli (RoBERTa backbone) scores the text against all four candidate labels simultaneously.

03
Pattern blending

Scores are blended 70/30 with a rule-based regex matcher trained on curated gaslighting and sarcasm phrase libraries.

04
Support output

For every flagged pattern: a sanity check, coping strategies, suggested responses, and relevant mental-health resources.

Model Architecture

Zero-shot classification with facebook/bart-large-mnli — a RoBERTa-based model trained on the MultiNLI corpus, repurposed here for multi-label intent classification without any task-specific fine-tuning.

Features

More than a
classifier

Every detection comes with actionable support — not just a label and a score.

💬
Single Message Analysis

Paste any message and get an instant breakdown — label, confidence score, flagged phrases, and support resources.

🔗
Conversation Thread Audit

Analyze up to 5 messages as a window to detect escalation patterns and dominant toxicity over time.

🧪
Stress Test Mode

Run a built-in evaluation suite of positive-worded insults. Pass threshold: ≥70% correctly flagged as Sarcastic.

🧠
Sanity Checks

"You are NOT being overly sensitive." — for every toxic detection, a grounding statement to validate the reader's experience.

🛠️
Response Scripts

Suggested responses tailored to the detected pattern — calm, boundary-setting language you can use immediately.

📋
Coping Strategies

Evidence-based coping techniques matched to severity level, from grounding exercises to seeking professional support.
